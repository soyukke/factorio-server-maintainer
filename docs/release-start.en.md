# Getting Started

1. Extract this zip into a writable folder.  
   Example: `%USERPROFILE%\Apps\FactorioServerMaintainer`
2. Run `factorio-server-manager.exe`.
3. On first launch, press `Save` to create the app config.
4. Press `Update` to install SteamCMD and the Factorio server.
5. Press `Start` after the server is installed.

You do not need `just setup` or a Rust toolchain.

If SteamCMD asks you to log in, enter your Steam username, password, and Steam Guard code in the console window.
Your password is not stored in this app's config file.

See `docs\user-guide.en.md` for the full guide.
