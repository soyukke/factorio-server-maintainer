# はじめに

1. このzipを、書き込み可能な場所に展開します。  
   例: `%USERPROFILE%\Apps\FactorioServerMaintainer`
2. `factorio-server-manager.exe` を起動します。
3. 初回は `保存` を押して設定を作成します。
4. `更新` を押すと SteamCMD と Factorio server を取得します。
5. 取得後に `開始` を押すとサーバーが起動します。

`just setup` や Rust toolchain は不要です。

SteamCMD がログインを求めた場合は、表示されたコンソールで Steam ユーザー名、パスワード、Steam Guard code を入力してください。
パスワードはこのアプリの設定ファイルには保存されません。

詳しい使い方は `docs\user-guide.ja.md` を見てください。
